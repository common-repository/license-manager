== Description ==

License manager for digital product online or remote activation.

[Client Side Demo Plugin](https://pickplugins.com/docs/documentation/license-manager/client-side-demo-plugin/)


<strong>Translation</strong>

Pluign is translation ready , please find the 'en.po' for default translation file under 'languages' folder and add your own translation. you can also contribute in translation, please contact us http://www.pickplugins.com/contact/



== Installation ==

1. Install as regular WordPress plugin.<br />
2. Go your plugin setting via WordPress Dashboard and find "<strong>License Manager</strong>" activate it.<br />
